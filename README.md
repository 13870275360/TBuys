# TBuys #

用户购物电商网站：Node.js+Express+MongoDB+Mongoose

## 步骤 ##


> 
1. `git clone path`下载项目到本地；
2. `npm install`安装依赖和相关插件;
3. 安装`MongoDB`,设定数据库存放位置`mongod --dbpath "\MondoDB\data"`可以自主设定，启动数据库；
4. 运行项目`node app.js`,打开浏览器进行测试，默认端口号`80`，可自由设定监听端口
